조건문

        - 프로그래밍 코드가 해석되는 순서를 제어하는 제어문 중 한 종류
        - 작성된 조건식에 따라서 수행되는 코드를 변경하는 구문
            * 조건식 : 결과가 true 또는 false인 식
                ex) 논리 연산자(&&, ||, !)
                    비교 연산자(>, ==, !=, ...)
                    함수 호출결과로 boolean 반환
                    boolean 단독 작성

if문 (만약에)

        
            if(조건식) {
                조건식이 true인 경우 수행할 코드;
            }

입력받은 수가 양수인지 확인

if-else문 (만약에 ~ 아니면)

        
            if(조건식) {
                조건식이 true인 경우 수행할 코드;
            }
            else {
                조건식이 false인 경우 수행할 코드;
            }

            * else는 조건식을 작성하지 않는다

1~100 사이의 난수를 발생시켜 홀/짝 판별하기

if-else if-else문

        
            if(조건식) {
                조건식이 true인 경우 수행할 코드;
            }
            else if(조건식) {
                앞선 if문이 false이면서 현재 else if문이 true인 경우 수행할 코드;
            }
            else {
                앞선 모든 if문이 false인 경우 수행할 코드;
            }

Early Return

        return;     : 현재 실행중인 함수를 "즉시 종료"하고 호출한 곳으로 돌아감
        
        if문 조건에 현재 수행하려는 기능의 예외 상황을 조건식으로 작성하여
        예외 상황 발생 시 즉시 함수를 종료하게 만드는 방법

        - 장점 : 코드 들여쓰기가 줄어들고 간결해짐
                 유지보수성 향상
        - 단점 : return 구문에 함수 내에 분산되어 함수 코드의 복잡도가 늘어남

switch, case, default

        switch문
        - 식 하나의 결과로 많은 경우의 수를 처리할 때 사용하는 조건문
        - 식의 결과와 일치하는 값을 가진 case 구문이 수행됨
            == if/else if(A===B) 형태와 유사
        - 식의 결과와 일치하는 case가 없을 경우 default 구문 수행
            == else와 유사

        
            switch(식){      // 식의 결과값은 정수 또는 문자열
                case 결과1 : 결과1의 수행코드; break;
                case 결과2 : 결과2의 수행코드; break;
                case 결과3 : 결과3의 수행코드; break;
                default;     // 모든 case가 만족하지 못하는 경우 수행
            }

입력된 수가 3,4,5가 맞는지 확인
